Standalone desktop applications for Windows and macOS. No browser needed – just download and run.

### macOS
[⬇️ Download CEBT-GUI.app](./macos/CEBT-GUI.app) (Intel & Apple Silicon)

---

## 🚀 Quick Start

### Windows
1. Download `CEBT-GUI.exe`
2. Double-click to run
3. Click "I Agree" (first time only)
4. Charge laptop to 100%, unplug
5. Click START TEST
6. Keep the app open (you can minimize)
7. Test stops automatically at 5%
8. Report saves to your chosen folder

### macOS
1. Download `CEBT-GUI.app`
2. Right-click and select "Open" (first time only)
3. Click "Open" in the security dialog
4. Follow same steps as Windows

## ❓ Frequently Asked Questions

**Q: Do I need to install anything?**  
A: No. Just download and run. The app is self-contained.

**Q: Is it safe?**  
A: Yes. The app runs entirely on your device. No data is sent anywhere.

**Q: Why does macOS show a security warning?**  
A: The app is not signed with an Apple developer certificate. Right-click and select "Open" to run it.

**Q: Where are reports saved?**  
A: You choose a folder when you start the test. If you don't choose, they save to your Documents/CEBT-Reports folder.

**Q: Can I close the app during test?**  
A: No – the test stops if you close the app. But it saves progress and will ask to resume when you reopen.

## ⚠️ Troubleshooting

### Windows
| Problem | Solution |
|---------|----------|
| App won't start | Download again – file may be corrupted |
| Antivirus warning | This is a false positive. Allow the app to run |
| Wake Lock not working | Make sure you're running on battery power |

### macOS
| Problem | Solution |
|---------|----------|
| "App is damaged" warning | Right-click the app and select "Open" |
| Can't open at all | Go to System Settings > Privacy & Security and click "Open Anyway" |
| App closes immediately | Check that your Mac meets minimum requirements |

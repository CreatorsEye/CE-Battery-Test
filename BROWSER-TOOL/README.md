## 📥 Download
[⬇️ Click here to download battery-reporter.html](./battery-reporter.html) (right-click → Save As)

## 🚀 Quick Start
1. **Open the file** in Chrome, Edge, Firefox, or Safari
2. **First time only**: Click "I Agree" to accept terms
3. **Charge your laptop** to 100%
4. **Unplug charger**
5. Click **START TEST**
6. **Use your laptop normally** (keep this tab open)
7. **Test stops automatically** at 5% battery
8. **Report saves** to your chosen folder (or Downloads)

---

## ✨ Features
| Feature | Description |
|---------|-------------|
| **Universal** | Works in all modern browsers |
| **Wake Lock** | Prevents sleep during test |
| **5-min logging** | Automatic every 5 minutes |
| **Auto-stop** | Stops at 5% to protect battery |
| **Charts** | Visual discharge curve |
| **Comparison** | Compare test durations |
| **Dark mode** | Matches your system theme |
| **Folder choice** | Pick where reports go (ZIP fallback) |
| **Recovery** | Resume if tab crashes |

## ❓ FAQ
**Q: Does it work in Firefox?**  
A: Yes! Full support with ZIP download fallback.

**Q: Will my computer sleep during test?**  
A: No – Wake Lock keeps it awake (audio fallback in Firefox).

**Q: Where are reports saved?**  
A: You choose a folder, or Downloads if unsupported.

**Q: Can I close the tab?**  
A: No – test stops if tab closes. But we save state for recovery.

---

## ⚠️ Troubleshooting
| Problem | Solution |
|---------|----------|
| Battery not showing | Use Chrome/Edge for best support |
| Wake Lock inactive | Firefox uses audio fallback (still works) |
| Folder picker missing | Firefox saves as ZIP – just as good |
| Test interrupted | Reopen – we'll ask to resume |
